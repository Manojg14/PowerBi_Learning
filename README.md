# 📊 Superstone Sales Dashboard (Power BI)

## Overview
The **Superstone Sales Dashboard** is an interactive Power BI project designed to analyze and visualize sales performance across categories, regions, and states.  
It transforms raw business data into actionable insights for better decision-making.

---

## Objectives
- Evaluate overall sales and profit performance  
- Identify top and underperforming sub-categories  
- Analyze regional and state-level trends  
- Enable interactive data exploration  

---

## Key Metrics
| Metric            | Value   |
|------------------|--------|
| Total Sales      | 2.30M  |
| Total Profit     | 286.40K|
| Total Quantity   | 38K    |
| Total Orders     | 9,994  |

---

## Features

### 🔹 Sales Insights
- Category-wise sales distribution  
- Sub-category performance comparison  

### 🔹 Profit Analysis
- Region-wise profit breakdown  

### 🔹 Performance Tracking
- Top 3 sub-categories by sales  
- Bottom 3 sub-categories by sales  

### 🔹 Geographic Analysis
- State-wise sales and profit visualization  

### 🔹 Interactivity
- Dynamic slicers (Sub-Category, Order Date)  
- User-friendly and responsive design
